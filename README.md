### 1.2 Arbeitspakete

| Nr. | Typ | Verbindlichkeit | Beschreibung |
| --- |---- | ----- | --------- |
| 1.| muss | Funktional | Die Website ist erstellt. |

| 5.| kann | Qualitativ | Die Website hat eine Überuns-seite |
